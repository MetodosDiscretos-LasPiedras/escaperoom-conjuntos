Escape Room: Métodos Discretos - UTU 🎓🔐

Este es un proyecto educativo interactivo diseñado para estudiantes de 1º de Bachillerato de la UTU. Se trata de un "Escape Room" matemático enfocado en la Teoría de Conjuntos, donde los alumnos deben resolver desafíos lógicos para recuperar la contraseña del WiFi del instituto.

🚀 Características Principales

Desafíos Aleatorios: Cada sesión genera valores y conjuntos distintos. Los estudiantes no pueden simplemente copiarse los resultados, ya que deben razonar cada problema.

Sistema de Puntuación: - Se inicia con 1000 puntos.

Cada error en una pista penaliza con -50 puntos.

Los errores al intentar adivinar la frase final restan -20 puntos.

Gamificación: Al finalizar, el sistema otorga un Rango de Especialista basado en el puntaje obtenido (desde "Usuario en Prácticas" hasta "Analista de Élite").

Privacidad y Registro: Solo solicita Nombre y Apellido para personalizar la experiencia y el diploma final.

Tecnología: Desarrollado con React, Tailwind CSS y Lucide Icons en un único archivo autónomo.

📚 Contenido Matemático

El juego cubre los siguientes temas clave de Métodos Discretos:

Unión y Cardinalidad ($|A \cup B|$)

Determinación por Comprensión (Rangos en $\mathbb{N}$ y números pares)

Complemento de un Conjunto ($A'$) dado un Universo

Diferencia Simétrica ($A \Delta B$)

Propiedades e Inclusión ($B \subseteq A$, Intersección e Inclusión)

Producto Cartesiano ($A \times B$)

Diferencia de Conjuntos ($A - B$)

Determinación por Extensión (Vocales y filtrado de elementos)

🛠️ Instalación y Uso

Para usar este escape room con tus alumnos, no necesitas instalar nada complicado:

Crea un repositorio en GitHub.

Sube el archivo index.html a la raíz del repositorio.

Ve a Settings > Pages.

En la sección Build and deployment, selecciona la rama main y la carpeta /(root).

¡Listo! GitHub te dará un enlace público para compartir con tu clase.

📖 Frase de Acceso Final

La frase que los alumnos deben reconstruir es:

"La lógica de conjuntos es la base digital"
